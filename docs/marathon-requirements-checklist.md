## ✅ Вимога 3: Тема проєкту відповідає темі Марафону

### Тема: **Secret Nick (Таємний Святий Миколай)**

**Призначення:** Веб-застосунок для організації таємного обміну подарунками (Secret Santa) в групах людей.

**Функціональність:**
- Створення "кімнат" для груп людей (офісні команди, друзі, родина)
- Управління учасниками обміну подарунками
- Встановлення бюджету подарунків
- Управління списками бажань
- Інформація про адреси доставки
- Облік особливих побажань учасників (сюрпризи, уваги)

**Алгоритм розподілу подарунків:** Secret Santa забезпечує випадковий розподіл — кожен учасник отримує ім'я іншого учасника для покупки подарунка.

**Очікувані сценарії використання:**
1. Адміністратор створює кімнату обміну
2. Запрошує учасників за унікальним кодом
3. Кожен учасник попадає свої контакти та список бажань
4. Система розраховує, хто кому дарує подарунок
5. Учасники отримують інформацію про адресата та їх уваги

## ✅ Вимога 4: Опис додаткової реалізованої функціональності

### Детальний опис див. у файлі: [`docs/additional-functionality.md`](./additional-functionality.md)

**Основні реалізовані особливості:**

### 1️⃣ **Мультиплатформена фронтенд-розробка**
- React з Vite для швидкої розробки
- Angular як альтернативна реалізація
- SCSS для стилізації з підтримкою теми
- Responsive дизайн для всіх пристроїв

### 2️⃣ **Масштабована архітектура бекенду**
- 4-шарова архітектура (.NET: Api → Application → Domain → Infrastructure)
- REST API з повною CRUD функціональністю
- Валідація на 2му рівні складності (9+ перевірок)
- FluentValidation для всіх моделей

### 3️⃣ **Хмарна інфраструктура як код (Terraform)**
- 5 модулів Terraform (VPC, Security Groups, EC2, RDS, ALB)
- GitHub Actions автоматизація розгортання
- AWS Certificate Manager для SSL/TLS управління
- Масштабованість та high availability

### 4️⃣ **Docker контейнеризація**
- Docker Compose для локальної розробки з 4 сервісами
- Multi-stage builds для оптимізації образів
- Health checks та управління
- Підтримка оточення для розробки/продакшену

### 5️⃣ **Автоматизовано тестування**
- BDD фреймворк (Reqnroll + NUnit)
- API тести для всіх ендпоінтів
- UI тести з використанням Selenium
- Інтеграція в CI/CD pipeline

### 6️⃣ **Безпека**
- HTTPS/SSL-TLS шифрування
- Унікальні користувацькі коди замість простих ID
- Ролева авторизація (адміністратор vs учасник)
- SQL Injection захист через ORM
- CORS конфігурація

### 7️⃣ **Документація**
- Swagger/OpenAPI для API
- Комплексні README файли для кожного компоненту
- Інструкції для локальної розробки та розгортання
- Приклади API запитів

### 8️⃣ **Хмарна готовність**
- Автоматичне масштабування з ALB
- Multi-AZ розгортання RDS
- Резервне копіювання та моніторинг
- Health checks на всіх рівнях
